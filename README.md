# forest-kingdoms-rpg

3D-экшн игра с возможностью играть за лесных эльфов, охрану дворца или злодея. Игра включает четыре зоны, систему повреждений персонажа и элементы RPG.

## Описание игры

Forest Kingdoms RPG - это захватывающая 3D-игра, в которой игроки могут выбирать между тремя уникальными фракциями: лесными эльфами, охраной дворца и злодеями. Каждая фракция предлагает свой уникальный геймплей и цели.

## Игровые зоны

Игра разделена на четыре основные зоны:
1. Лес Эльфов - живописная зона с магическими существами и древними руинами
2. Дворцовый город - величественная столица с роскошными зданиями и садами
3. Подземелья - темные и опасные лабиринты под дворцом
4. Запретный храм - таинственное место с могущественными артефактами

## Система повреждений

Реалистичная система повреждений персонажа:
- Потеря конечностей (руки, ноги)
- Повреждения органов чувств (глаза, уши)
- Ранения различной степени тяжести
- Возможность лечения у лекаря или покупки протезов

## Особенности RPG

- Развитие персонажа и навыков
- Система квестов и заданий
- Взаимодействие с NPC
- Сбор ресурсов и крафт предметов

## Процедурная генерация

Forest Kingdoms RPG использует продвинутую систему процедурной генерации для создания уникального игрового опыта при каждой новой игре:

### Генерация мира
- Использование шума Перлина для создания разнообразного ландшафта
- Процедурная генерация биомов (леса, равнины, горы, водоемы)
- Случайное размещение растительности и природных объектов

### Архитектура и здания
- Алгоритмы генерации зданий, соответствующих стилю каждой фракции
- Процедурная генерация интерьеров с уникальной планировкой
- Случайное размещение городских кварталов и построек

### Персонажи и существа
- Случайная генерация внешности, имен и характеристик NPC
- Процедурное определение личности и поведения персонажей
- Генерация групп NPC с учетом фракционной принадлежности

### Квесты и задания
- Шаблонная система квестов с случайными параметрами
- Процедурная генерация целей, наград и локаций заданий
- Цепочки квестов, адаптирующиеся под уровень игрока

### Предметы и снаряжение
- Система генерации предметов с различным качеством (обычные, редкие, эпические, легендарные)
- Процедурное определение характеристик, эффектов и названий предметов
- Случайные наборы лута, соответствующие уровню игрока

## Сетевая архитектура

Forest Kingdoms RPG использует децентрализованную P2P архитектуру для мультиплеерного режима, что позволяет игрокам напрямую подключаться друг к другу без центрального сервера.

### Основные особенности:
- **Децентрализованная сеть**: Отсутствие необходимости в центральном сервере
- **Масштабируемость**: Распределение нагрузки между всеми участниками
- **Отказоустойчивость**: Продолжение игры даже при выходе некоторых участников
- **Гибкость**: Поддержка различных режимов игры (кооператив, PvP, песочница)

### Компоненты сетевой системы:
- **P2P Фреймворк**: Универсальный фреймворк для создания децентрализованных мультиплеерных приложений
- **Обнаружение узлов**: Автоматическое обнаружение доступных сессий
- **Управление соединениями**: Надежное установление и поддержание соединений
- **Обмен сообщениями**: Эффективная передача игровых данных
- **Управление сессиями**: Создание и управление игровыми сессиями
- **Обработка ошибок**: Механизмы восстановления после сетевых сбоев

## Система локализации

Forest Kingdoms RPG поддерживает несколько языков (русский, английский, китайский) с возможностью добавления новых языков. Подробнее о системе локализации см. в [документации по локализации](docs/localization.md).

## Техническая информация

### Игровой движок
Проект разработан с использованием Godot Engine 4.x

### Структура проекта
- `src/scenes` - директория для сцен игры
- `src/scripts` - директория для скриптов GDScript
- `src/resources` - директория для ресурсов игры
- `src/autoload` - директория для автоподгружаемых скриптов
- `src/generation` - директория для скриптов процедурной генерации
- `src/networking` - директория для P2P сетевого фреймворка
- `assets` - директория для хранения ассетов (модели, текстуры, звуки)
- `docs` - директория с документацией проекта
- `config` - директория с конфигурационными файлами
- `locale` - директория с файлами локализации

### Настройка проекта
1. Установите Godot Engine 4.x
2. Откройте проект через файл `project.godot`
3. Для запуска игры нажмите кнопку "Play" в редакторе Godot
