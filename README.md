# Forest Kingdoms RPG

## Описание

Forest Kingdoms RPG - это захватывающая 3D-игра с открытым миром, в которой игроки могут выбирать между тремя уникальными фракциями: лесными эльфами, охраной дворца и злодеями. Каждая фракция предлагает свой уникальный геймплей и цели.

Игра отличается реалистичной системой повреждений персонажа, процедурной генерацией мира и децентрализованной P2P архитектурой для мультиплеерного режима. Проект разрабатывается с использованием Godot Engine 4.x (основная версия) и может работать с Godot Engine 3.x (альтернативная версия для совместимости).

## Основные особенности

### Три уникальные фракции
- **Лесные эльфы** - защитники природы с магией природы
- **Охрана дворца** - представители закона и порядка
- **Злодеи** - антагонисты, стремящиеся захватить власть

### Реалистичная система повреждений
- Потеря конечностей (руки, ноги) с возможностью протезирования
- Повреждения органов чувств (глаза, уши) с долгосрочными последствиями
- Система лечения и восстановления

### Четыре игровые зоны
1. Лес Эльфов - живописная зона с магическими существами
2. Дворцовый город - величественная столица с роскошными зданиями
3. Подземелья - темные лабиринты под дворцом
4. Запретный храм - таинственное место с артефактами

### Процедурная генерация
- Ландшафта и растительности
- Зданий и сооружений
- NPC и существ
- Квестов и предметов

### Децентрализованная P2P архитектура
- Мультиплеерный режим без центрального сервера
- Распределенная нагрузка между участниками
- Отказоустойчивость сети
- Интеграция JavaScript P2P компонентов с Godot

### Интеграция с социальными сетями
- Поддержка авторизации через популярные социальные сети
- Возможность делиться достижениями и скриншотами
- Синхронизация игрового прогресса между устройствами
- Интеграция с внутриигровыми сообществами

### Текстовый MUD режим
- Альтернативный текстовый интерфейс для игры
- Поддержка всех основных игровых функций
- Совместимость с основной 3D версией игры
- Подробнее в [документации по MUD режиму](docs/mud-mode.md)
