# Система локализации Forest Kingdoms RPG

## Обзор

Этот документ описывает систему локализации Forest Kingdoms RPG, которая позволяет поддерживать несколько языков в игре. Система основана на CSV-файлах с переводами и автозагружаемом скрипте LocalizationManager.

Игра основана на мемах, поэтому система локализации учитывает особенности перевода мемных элементов. Мемы часто уникальны для каждой культуры и не могут быть просто переведены с одного языка на другой. Вместо этого для каждого языка/культуры могут потребоваться свои уникальные мемы.

## Структура каталогов

```
res://locale/
├── translations.csv    # Основной файл с переводами
```

## Формат файла переводов

Файл `translations.csv` содержит строки для перевода в формате CSV:

```
id,ru,en,zh
GAME_LOADED,Forest Kingdoms RPG - Игра загружена,Forest Kingdoms RPG - Game loaded,森林王国 RPG - 游戏已加载
MEME_EXAMPLE_1,Когда ты выбрал лесных эльфов и понял что это мем,"When you chose forest elves and realized it's a meme","当你选择了森林精灵并意识到这是个梗"
```

- `id` - уникальный идентификатор строки
- `ru` - русский перевод
- `en` - английский перевод
- `zh` - китайский перевод

## Добавление новых строк

1. Добавьте новую строку в файл `locale/translations.csv`
2. Укажите идентификатор и переводы на все поддерживаемые языки
3. Используйте идентификатор в коде через `LocalizationManager.get_string("ID")`

Пример:
```gdscript
print(LocalizationManager.get_string("NEW_STRING_ID"))
```

## Локализация мемов

Поскольку игра основана на мемах, важно понимать особенности их локализации:

1. **Культурная уникальность**: Мемы часто уникальны для каждой культуры. Вместо прямого перевода мема, может потребоваться замена его на эквивалентный мем в целевой культуре.

2. **Контекстуальная адаптация**: Мемы могут требовать адаптации под контекст игры. Например, мем, связанный с поп-культурой одной страны, может быть заменен на аналогичный мем, понятный другой аудитории.

3. **Визуальные мемы**: Для визуальных мемов (изображений) может потребоваться создание отдельных версий для каждой локали.

4. **Идентификаторы мемов**: Используйте специальные идентификаторы для мемных элементов, например, с префиксом `MEME_`, чтобы их было легче находить и управлять ими.

Пример локализации мема:
```
MEME_FOREST_ELVES,Когда ты выбрал лесных эльфов и понял что это мем,"When you chose forest elves and realized it's a meme","当你选择了森林精灵并意识到这是个梗"
```

## Добавление новых языков

1. Добавьте новый столбец в файл `locale/translations.csv` с кодом языка (например, `es` для испанского)
2. Добавьте код языка в массив `supported_languages` в скрипте `src/autoload/LocalizationManager.gd`
3. Заполните переводы для нового языка в CSV-файле
4. Для мемных элементов учитывайте культурные особенности целевой аудитории

## API LocalizationManager

### `set_language(lang)`

Устанавливает текущий язык.

Параметры:
- `lang` (String) - код языка (например, "ru", "en", "zh")

### `get_string(id)`

Возвращает переведенную строку по идентификатору.

Параметры:
- `id` (String) - идентификатор строки

Возвращает:
- (String) - переведенная строка или идентификатор, если перевод не найден

### `get_supported_languages()`

Возвращает список поддерживаемых языков.

Возвращает:
- (Array) - массив кодов поддерживаемых языков
