# Обзор P2P функциональности

Игра Forest Kingdoms RPG интегрирована с децентрализованным P2P компонентом, который предоставляет следующие возможности:

1. **Синхронизация состояния игры** между узлами с использованием CRDT (Conflict-free Replicated Data Types)
2. **Идентификация узлов** через DID (Decentralized Identifiers) и Nostr ключи
3. **Безопасная передача данных** с криптографической привязкой идентификаторов

## Основные компоненты

### IdentityLayer
Слой идентификации, реализующий DID документы на основе ключей и привязку к Nostr публичным ключам.

### StateSyncLayer
Слой синхронизации состояния, использующий CRDT для обеспечения консистентности данных между узлами.

### CRDT реализации
Конкретные реализации CRDT, такие как LWWRegister (Last-Write-Wins Register) и ORSet (Observed-Remove Set).

## Интеграция с игровым движком

Интеграция осуществляется через P2PAdapter, который предоставляет интерфейс между JavaScript P2P компонентами и игровым движком Godot.
