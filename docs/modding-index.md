# Документация по моддингу Forest Kingdoms RPG

Добро пожаловать в документацию по системе моддинга Forest Kingdoms RPG. Эта система позволяет создавать и интегрировать пользовательские компоненты в игру без необходимости изменения основного кода.

## Оглавление

1. [Введение в моддинг](modding.md)
   - Описание системы моддинга
   - Цели и компоненты
   - Требования к реализации

2. [API для моддинга](modding-api.md)
   - Основные компоненты API
   - Методы и функции
   - Безопасность выполнения модов

3. [Руководство по созданию модов](modding-tutorial.md)
   - Подготовка к разработке
   - Структура мода
   - Пошаговое создание простого мода
   - Продвинутые возможности

4. [Примеры модов](modding-examples.md)
   - Простые моды добавления контента
   - Моды, изменяющие игровую механику
   - Моды с пользовательским интерфейсом
   - Моды, добавляющие новые локации и квесты

## Начало работы

Для начала создания модов рекомендуем следующий порядок изучения материалов:

1. Ознакомьтесь с [введением в моддинг](modding.md)
2. Изучите [руководство по созданию модов](modding-tutorial.md)
3. Посмотрите [примеры модов](modding-examples.md) для понимания возможностей
4. Используйте [документацию API](modding-api.md) для создания собственных модов
