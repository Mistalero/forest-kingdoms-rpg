# Интеграция с социальными сетями

## Обзор

Этот документ описывает реализацию интеграции с социальными сетями в игре Forest Kingdoms RPG. Интеграция позволяет игрокам авторизоваться через популярные социальные сети, делиться достижениями и синхронизировать игровой прогресс между устройствами.

## Поддерживаемые платформы

На данный момент поддерживаются следующие социальные сети:
- Mail.ru
- ВКонтакте (VK)

## Архитектура

Интеграция реализована в виде модуля в директории `src/networking/social-networks-integration/` и включает следующие компоненты:

### social_networks_api.gd
Основной класс для взаимодействия с API социальных сетей. Реализует функции:
- Инициализация подключения
- Аутентификация пользователей
- Обмен данными

### social_networks_config.gd
Конфигурационный файл, содержащий параметры подключения к различным социальным сетям.

### oauth_handler.gd
Обработчик OAuth авторизации, управляющий процессом аутентификации пользователей.

## Процесс авторизации

1. Пользователь выбирает социальную сеть для авторизации
2. Открывается окно браузера с запросом на авторизацию
3. После подтверждения пользователь перенаправляется обратно в игру
4. Игра получает токен доступа и данные пользователя
5. Пользователь считается авторизованным

## Синхронизация данных

Игра может синхронизировать следующие данные с профилем пользователя в социальной сети:
- Достижения и награды
- Статистика игрового прогресса
- Скриншоты и медиафайлы
- Статус онлайн

## Безопасность

Все токены доступа хранятся в зашифрованном виде и передаются по защищенным каналам связи. Персональные данные пользователей не передаются третьим сторонам без их согласия.

## Расширение функциональности

Для добавления поддержки новых социальных сетей необходимо:
1. Добавить конфигурацию новой сети в `social_networks_config.gd`
2. Реализовать специфичные для этой сети методы в `social_networks_api.gd`
3. Обновить документацию

## API Reference

### SocialNetworksAPI

#### initialize_connection(network_name: String) -> bool
Инициализирует подключение к указанной социальной сети.

#### authenticate_user(network_name: String, credentials) -> bool
Аутентифицирует пользователя в указанной социальной сети.

#### share_content(network_name: String, content: Dictionary) -> bool
Публикует контент в указанной социальной сети.

### SocialNetworksConfig

#### get_network_config(network_name: String) -> Dictionary
Возвращает конфигурацию для указанной социальной сети.

#### set_client_credentials(network_name: String, client_id: String, client_secret: String)
Устанавливает учетные данные клиента для указанной социальной сети.

### OAuthHandler

#### start_oauth_flow(network_name: String, config: Dictionary)
Запускает процесс OAuth авторизации для указанной социальной сети.

#### handle_oauth_callback(network_name: String, callback_data: Dictionary)
Обрабатывает обратный вызов OAuth для указанной социальной сети.

#### refresh_access_token(network_name: String, refresh_token: String) -> String
Обновляет токен доступа для указанной социальной сети.

#### validate_access_token(network_name: String, access_token: String) -> bool
Проверяет действительность токена доступа для указанной социальной сети.
