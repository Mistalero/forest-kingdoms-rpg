# Правила для SourceCraft Assistant

Этот файл содержит правила и ограничения для работы SourceCraft Assistant с данным проектом.

## Запрещенные действия

1. **НЕ ИЗМЕНЯТЬ файл README.md ни при каких обстоятельствах**
   - Только пользователь catman78 имеет право редактировать README.md
   - Любые изменения README.md строго запрещены для ассистента

## Разрешенные действия

1. Все остальные файлы проекта можно изменять по необходимости
2. Можно создавать новые файлы в соответствии с задачами проекта
3. Можно обновлять существующие файлы документации (кроме README.md)
4. Можно вносить изменения в кодовую базу по мере необходимости

## Исключения

1. Если явно указано в задаче, что нужно изменить README.md, то такое изменение возможно только после подтверждения пользователем catman78

## Контакты

При возникновении вопросов по данным правилам обращайтесь к пользователю catman78.
